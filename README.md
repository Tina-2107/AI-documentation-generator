AI Documentation Generator

An AI-powered application that automatically analyzes source code and generates high-quality technical documentation using Large Language Models (LLMs), Python AST parsing, and Retrieval-Augmented Generation (RAG).

🚀 Project Goal

The goal of this project is to simplify software documentation by automatically generating:

README files
Function documentation
Class documentation
API documentation
Project summaries
Code explanations

Instead of manually writing documentation, developers can upload their project and receive AI-generated documentation.
